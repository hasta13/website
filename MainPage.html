Shared Desktop Setup Using VNC

  • Setup VNC server
    1. Install VNC server here: VNC Server Install File
    2. Get public IP address
    3. (Optional) Setup port forwarding on router (3389 for RDP, or 5900 for DS)
  • Setup VNC Viewer (remote computer)
    1. Install VNC Viewer VNC Viewer Install File
    2. Use VNC server public IP address found above

Useful tricks for Windows

  • Managing start up programs: start -> shell:startup
    • Or to configure ALL windows start up programs: start -> msconfig
  • Useful hardware information about a computer: start -> dxdiag

Github push

  • Create a new respository on the command line:
    1. touch README.md
    2. git init
    3. git add README.md
    4. git commit -m "first commit"
    5. git remote add origin https://github.com/hasta13/tricks.git
    6. git push -u origin master

  • Push an existing repository from the command line:
    1. git remote add origin https://github.com/hasta13/tricks.git
    2. git push -u origin master
